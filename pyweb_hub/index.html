What is PyWeb?

PyWeb is an echo-system dedicated to the wed world with python and only python!

With PyWeb you get:

  • A new DOM system writen with Python
  • PyWeb Client - a browser, writen with - guess what? - python, that allow you to browse websites build with PyWeb
  • PyWeb Hub -
  • PyWebAct - React-like framework with PyWeb
